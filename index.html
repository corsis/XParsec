XParsec

A stream-type-independent parsec implementation in F# 3.0 - inspired by FParsec, freed from CharStreams and simplified.

View project onGitHub

Example

open XParsec.Combinators
open XParsec.Xml

let s = AE.New [| X.Parse "<span font='Bold' />"; X.Parse "<span t='(' />" |]
let r = next >. !@"font"|->String.length .> "font"@~?"B" .>. next .> !@+"t" <| s

val r : (int * X) Reply = S (4, <span t="(" />)
